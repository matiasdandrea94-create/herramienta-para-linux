La herramienta es 100% Python estándar, sin dependencias externas. Tiene los mismos 6 módulos de antes pero interactivos en la consola:

nmap helper — te pide el target, elegís un preset y podés ejecutar el comando directamente desde la tool
subdominios — genera comandos para subfinder, amass, ffuf con pipeline incluido
SQLi payloads — 6 modos (detect, union, blind, sqlmap, bypass, error-based) con descripciones
bruteforce — hydra, ffuf, gobuster y medusa con selector de wordlist
OWASP checklist — interactiva con toggle y barra de progreso
